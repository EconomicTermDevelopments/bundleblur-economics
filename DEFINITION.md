# Bundleblur: Technical Definition

    ## Formal Definition
    Intentional obfuscation through complex bundling preventing price comparison.

    ## Conceptual Background
    Bundleblur is used here as an operational concept rather than a rhetorical label. It addresses strategic obfuscation through product/service bundling that impairs price comparison. The mechanism focus is complex package architecture raising search and cognitive costs to weaken consumer price discipline, which can be represented with explicit channels: bundle_complexity, opaque_fee_share, add_on_density, comparability_gap, search_cost, cognitive_load, disclosure_clarity. This repository couples concept definition with executable implementation so assumptions remain inspectable. In practice, the framework supports comparative statics, scenario simulation, and policy diagnostics. Typical applications include consumer-protection auditing, pricing transparency diagnostics, and digital marketplace design evaluation. Because the index logic is transparent, researchers can vary weights and test robustness across model specifications.
